## Summary

```
Traditional:  cost_per_call = O(depth × dim²)   ← 매 호출마다 N단계 순차 실행
AXOL Weave:   cost_one_time = O(depth × dim³)   ← 1회 직조 (행렬 합성 + 품질 보증)
AXOL Observe: cost_per_call = O(dim²)            ← depth 무관! 단일 행렬-벡터 곱
```

N회 관측 시:
```
Traditional = N × O(depth × dim²)
AXOL        = O(depth × dim³) + N × O(dim²)

N >> depth × dim 일 때 AXOL은 depth만큼 가속.
```
